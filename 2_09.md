# SQL SYNTAX

```
sql keyword는 대문자로 쓰자
sql statement 끝에 ;(세미콜론) 쓰기
```

# SQL statemtns 유형

```
DDL- 데이터 정의
      SQL 키워드: CREATE DROP ALTER
DQL- 데이터 검색
*중요 SQL 키워드: SELECT
DML- 데이터 조작
      SQL 키워드: INSERT UPDATE DELETE
DCL- 데이터 제어
      SQL 키워드: COMMIT ROLLBACK GRANT REVOKE
```

# SELECT syntax

```
SELECT
    select_list    (필드)
FROM
    table name:    (테이블)
```

# mysql

```
첫번째 번개는 전부 실행
두번째 번개는 cursor 있는 명령문 실행
```

# ORDER BY syntax

```
ASC:오름차순(기본 값)
DESC: 내림차순

정렬 두번쨰는 첫번째에서 겹치는거 중에서만 정렬
```

# SELECT statement 실행 순서

```
1 FROM
2 SELECT
3 ORDER BY
```
