# FRIDAY

## Branch

1. 브랜치 생성

```
(master)$ git branch (branch name)
```

2. 브랜치 이동

```
(master)$ git checkout (branch name)
```

3. 브랜치 생성 및 이동

```
(master)$ git checkout -b (branch name)
```

4. 브랜치 목록

```
(master)$ git branch
```

5. 브랜치 삭제

```
(master)$ git branch -d (branch name)
```

## Merge

```
각 branch에서 작업 한 이후 이력을 합치기 위해 사용
```

## Fork

```
원격 저장소를 fork한다.
내 저장소로 복제본을 가져옴으로써 로컬에서 작업 후 원격 저장소로 push가능
*clone시 반드시 본인 저장소인지 확인*
```
